Web Server Services

Monitors the web for client requests and serves back responses such as images, text, files, video, etc...

Examples of Web Servers:

  • IIS
  • Apache
  • NGINX

Web Application or Site

The logic, code, and buisiness rules

  • Can be static or dynamic
  • SharePoint

Server Side Language Interpreters

Used to interpret the code and frameworks used to create the site and do the “stuff” the code instructs

Examples of Server Side Languages:

  • PHP
  • Ruby
  • C#

Examples of Server Side Frameworks:

  • CakePHP
  • Rails
  • ASP.NET

Data Retention

Used to store user data between sessions

Examples of databases used:

  • PostgreSQL
  • MS SQL Server
  • MySQL
